stagewise is a desktop browser with a built-in AI coding agent. It connects to your local development environment, lets you browse your running app, select UI elements, and instruct the agent to make code changes — all without leaving the browser.

## Key capabilities

- **AI coding agent** — Chat with a powerful AI assistant that can read, write, and modify files in your workspace
- **DOM context selection** — Click on any element in the browser to attach it as context for the agent
- **Diff review** — Accept, reject, or partially accept code changes with full inline diffs
- **Workspace awareness** — The agent understands your project structure via auto-generated `WORKSPACE.md` and custom skills
- **BYOK support** — Use your own API keys for Anthropic, OpenAI, Google, and more
- **Plugin system** — Extend the agent with custom plugins and skills per workspace
