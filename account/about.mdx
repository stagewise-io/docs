Open the about page at **Settings → About**.

## Version information

The about page displays the current stagewise version and build information.

## Update channels

stagewise is distributed through two pre-release update channels:

| Channel | Description |
|---------|-------------|
| **Beta** | More stable pre-release builds. Recommended for most users. |
| **Alpha** | Bleeding-edge builds with the latest features. May contain bugs. |

### Changing the update channel

1. Open **Settings → About**
2. Under **Update Channel**, select **Alpha** or **Beta**
3. The change takes effect on the next update check

The current channel is inferred from the version string (versions containing `-alpha` default to the alpha channel).

## Open-source licenses

stagewise is built on open-source software. The about page includes a **Licenses** dialog that lists all third-party dependencies with:

- Package name and version
- License type (MIT, Apache-2.0, etc.)
- Publisher/author
- Full license text

Use the search bar within the licenses dialog to find specific packages.

stagewise itself is licensed under [AGPL-3.0](https://www.gnu.org/licenses/agpl-3.0.html).
