Open browsing settings at **Settings → Browsing → General**.

## Search engines

stagewise supports configurable search engines for the address bar. You can:

- **Set a default engine** — Used when you type a search query into the address bar
- **Add custom engines** — Define additional search engines with custom search URL patterns
- **Remove engines** — Delete search engines you don't need

### Adding a custom search engine

1. Click **Add search engine**
2. Enter:
   - **Name** — Display name (e.g. "DuckDuckGo")
   - **Search URL** — The URL pattern with `%s` as the query placeholder (e.g. `https://duckduckgo.com/?q=%s`)
3. Click **Save**

### Setting the default

Click the star icon next to any search engine to set it as the default. The default engine is used when you type a non-URL query into the address bar.
